Projects

ChatYourDocs

A user-friendly tool designed to elevate your interaction with ChatGPT by eliminating the concerns over its context window limitations. Simply upload your documents, and the app will facilitate detailed responses to your queries based on the content of the uploaded documents.

Whoop_GPT

This project aims to query data from Whoop fitness bands and provide insights and answers based on the user's data. We lean on Langchain as the application's orchestration layer, where we implement an Agent and custom Tools to be able to converse with the data. The application is built using FastAPI for backend services and Streamlit for the frontend.

ChatYourDocs x RAG

ChatYourDocs x RAG is an application that leverages Retrieval Augmented Generation (RAG) to query a vector database instead of using a relational database to store embeddings. It combines the power of language models with efficient vector retrieval to provide detailed responses based on document content.

DeepLearning.AI NLP Specialization
